# PHP Note

A CRUD Blogging Application to learn PHP and Symfony following Traversty Media's 'Up and Running with Symfony 4' series on Youtube.

This application uses:

* PHP 7
* Symfony 5 (Framework)
* Doctrine (Database Manager, ORM)
